# Shoresh UI Redesign — Design Spec

## Goal
Replace cold/neutral aesthetic with warm, inviting system using Fredoka + Nunito, sandy warm backgrounds, spacious schedule grid.

## Color System
- `--bg: #FAF6F0`, `--surface: #FFFCF8`, `--surface-elevated: #FFF8F0`
- `--border: #E8DDD0`, `--text: #2D1F12`, `--text-secondary: #7A6152`
- Keep: `--primary: #00ADBB`, `--warning: #F0585D`, `--success: #00AA59`

## Typography
- `--font-sans: Nunito`, `--font-condensed: Fredoka`, `--font-mono: IBM Plex Mono`

## Slot Cells
- `min-height: 56px`, `padding: 10px 12px`, `border-radius: 8px`
- Fill: `{color}1E` background, `1.5px solid {color}55` border

## Shared Styles
- `S.th`: Fredoka, uppercase, warm secondary color
- `S.btnPrimary/Secondary/Danger`: borderRadius 7px, Nunito 700
- `S.modalSm/Lg`: borderRadius 12px, warm surface-elevated
